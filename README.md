# Employee Management System

A Spring Boot REST API application for managing employee records using Java, Spring Boot, Spring Data JPA, and MySQL.

## Features

- Add Employee
- View Employee
- Update Employee
- Delete Employee
- Employee Data Validation
- Exception Handling
- MySQL Database Integration

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST APIs
- Git & GitHub

## Project Structure

- Controller Layer
- Service Layer
- Repository Layer
- Entity Layer
- Global Exception Handling
